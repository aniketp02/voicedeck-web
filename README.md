# VoiceDeck — Frontend

React 19 + TypeScript + Vite frontend for the VoiceDeck voice-interactive slide presenter.

See the [root README](../README.md) for setup and usage instructions.

See [`voicedeck-docs/technical/frontend.md`](../voicedeck-docs/technical/frontend.md) for full technical documentation.

## Development

```bash
npm install
npm run dev        # Start dev server on :5173 (proxies /ws and /api to :8000)
npm run build      # Production build
npm run lint       # ESLint
```
